Sorting-Algorithms
==================

Developed by <a href='https://github.com/LeonardoCardoso' target='_blank'>@LeonardoCardoso</a>. 

Quick set of sorting algorithms using Java.

<ul>
<li>Bubble Sort</li>
<li>Heap Sort</li>
<li>Insertion Sort</li>
<li>Merge Sort</li>
<li>Quick Sort</li>
<li>Selection Sort</li>
<li>Shell Sort</li>
</ul>
